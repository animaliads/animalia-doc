# Comportamentos

## Visão geral

Os comportamentos são padrões de práticas recomendadas sobre como uma pessoa usuária atinge um objetivo, mostrando combinações 
reutilizáveis de componentes e modelos mentais que atendam aos seus objetivos comuns com sequências e fluxos cujo o intuito 
seja alcançar uma boa usabilidade.

Esses padrões de comportamentos dos componentes estão alinhados com os [Princípios do **Animalia DS**](docs/about/principles), 
fornecendo uma linguagem unificada e aparência consistente ao projetar soluções dentro deste ecossistema.

### Lista de padrões

| Padrão                           | Objetivo                                                                                              |
| -------------------------------- | ----------------------------------------------------------------------------------------------------- |
| [`input`](input)                 | Inclusão de informações durante a utilização da solução.                                              |
| [`notification`](notification)   | Disponibilização de alertas ou aviso durante o fluxo de uma tarefa.                                   |
