# Motion
Refere-se a duração e suavização das animações entre os estados dos componentes.

## Duration
Refere-se ao tempo de transição de um ponto ao outro.

| Token                  | Valor (ms)  | 
|------------------------|-------------|
| `duration-extra-fast`  | 70 ms       |
| `duration-fast`        | 110 ms      |
| `duration-normal`        | 150 ms      |
| `duration-moderate`        | 240 ms      |
| `duration-slow`        | 400 ms      |
| `duration-extra-slow`        | 700 ms      |



## Timing

Refere-se ao suavição do movimento.

| Token                  | Valor                                       | 
|------------------------|---------------------------------------------|
| `timing-standart`      | Cubic-Bezier (0.35, 0.00, 0.10, 1.00)       |
| `timing-entering`      | Cubic-Bezier (0.30, 0.00, 1.00, 0.80)       |
| `timing-exiting`       | Cubic-Bezier (0.10, 0.00, 0.00, 1.00)       |
| `timing-continuos`     | Cubic-Bezier (0.00, 0.00, 0.00, 0.00)       |
