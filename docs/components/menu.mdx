## Visão geral

O componente `menu` tem como objetivo viabilizar a navegação e orienta a pessoa usuária dentro de uma interface.

## Anatomia

O componente foi estruturado em áreas para facilitar a estruturação das soluções.

### Estrutura macro

Em uma visão macro, o menu é composto pelas seguintes partes:
No formato Collapsed é composto por: Brand, Submenu e Action-Collapse e no formato extended apresenta Brand, Search, Submenu e Action-Collapse**, conforme imagem:**

<img src={MenuAnatomyMacro} alt="Menu macro" />

<img src={MenuAnatomy} alt="Menu base" />

1. **Brand**: Área reservada para a logo da solução;
2. **Submenu**: Apresenta elementos de que permite a navegação na interface;
3. **Action-Collapse**: Região de interação que permite variar entre as forma collapsed ou extended.
4. **Ani Search**
5. **Border**: forma que “emoldura” o conteúdo do componente
6. **Base**: fundo que encobre o conteúdo da página, trazendo a prioridade para o modal
7. **Ani Divider**
